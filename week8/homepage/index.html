Python

Python is a widely-used, high-level, general-purpose programming language known for its readability and simplicity, which emphasizes the use of significant indentation. Its design philosophy and extensive libraries make it versatile for various applications, from web development and data science to machine learning and automation.

Javascript

JavaScript (JS) is a high-level, multi-paradigm programming language and one of the core technologies of the World Wide Web, alongside HTML and CSS. It is primarily used to make web pages dynamic and interactive, but its usage has expanded significantly into server-side development, mobile apps, and more.

SQL

SQL (Structured Query Language) is a standard programming language used to define, manage, and retrieve data from relational databases. It is the most common method of interacting with data stored in tables (rows and columns) in systems like MySQL, PostgreSQL, Microsoft SQL Server, and Oracle.
